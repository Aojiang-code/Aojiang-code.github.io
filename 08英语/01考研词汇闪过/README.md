### 词根词缀记忆
使用词根词缀的方法来记忆单词可以帮助我们更好地理解和记忆它们的含义。以下是上述单词的巧记技巧，基于它们的词根和词缀：

1. **embody** - "em-" (进入) + "body" (身体) → 进入身体，即**体现**某种特质或理念。
2. **embrace** - "em-" (进入) + "brace" (臂弯) → 进入臂弯，即**拥抱**。
3. **enthusiasm** - "en-" (使成为) + "thusiasm" (源自希腊语，意为神灵附体) → 被激情充满，即**热情**。
4. **enthusiastic** - "en-" (使成为) + "thusiasm" (热情) + "-tic" (形成...的) → 充满热情的，即**热情的**。
5. **enlarge** - "en-" (使成为) + "large" (大的) → 使变大，即**扩大**。
6. **enhance** - "en-" (使成为) + "hance" (源自法语，意为优雅) → 使更加优雅或优越，即**提高**或**增强**。
7. **entitle** - "en-" (使成为) + "title" (标题，权利) → 给予标题或权利，即**命名**或**授权**。
8. **encounter** - "en-" (使成为) + "counter" (对抗) → 使对抗，即在对抗中**遭遇**或**遇到**。
9. **endure** - "en-" (使成为) + "dure" (持久) → 使持久，即**忍受**或**耐久**。
10. **react** - "re-" (再次) + "act" (行动) → 再次行动，即**反应**。
11. **reveal** - "re-" (打开) + "veal" (来自拉丁语，意为覆盖) → 打开覆盖，即**揭示**。
12. **retreat** - "re-" (向后) + "treat" (处理) → 向后处理，即**撤退**。
13. **reflect** - "re-" (回) + "flect" (弯曲) → 回弯曲，即**反映**。
14. **reflection** - "re-" (回) + "flection" (弯曲) + "-ion" (名词后缀，表示行为或状态) → 回弯曲的行为，即**反射**或**反思**。
15. **reinforce** - "re-" (再次) + "inforce" (加强) → 再次加强，即**加强**。
16. **remember** - "re-" (再次) + "member" (成员) → 再次成为成员，即**记得**。
17. **resort** - "re-" (再次) + "sort" (排序) → 再次排序，即**求助于**或**诉诸于**某种方法。
18. **register** - "re-" (回) + "gister" (记录) → 回记录，即**注册**或**登记**。
19. **recovery** - "re-" (回) + "covery" (覆盖) → 回覆盖，即**恢复**。
20. **replace** - "re-" (再次) + "place" (位置) → 再次放置，即**替换**。

通过词根词缀的记忆方法，我们可以更快地掌握单词的含义，并在实际使用中更加得心应手。
