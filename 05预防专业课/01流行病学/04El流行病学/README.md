**流行病学（Epidemiology）：**
流行病学是研究人群中**疾病**与**健康状况**的**分布**及其**影响因素**，并研究**防制疾病**及**促进健康**的**策略和措施**的科学。

- 原理：
  - 疾病与健康在人群中的**分布**；
  - 疾病的**发病过程**；
  - **人与环境**的关系；
  - **病因**论；
  - **病因推断**的原则；
  - 疾病防治的**原则和策略**；
  - 疾病发展的**数学模型**。

- 特征：
  - **群体**的特征；
  - **对比**的特征；
  - **概率论**和数理统计学的特征；
  - **社会心理**的特征；
  - **预防为主**的特征；
  - **发展**的特征。


- 挑战与展望：
  - **宏观**与微观并举；
  - **传染病**和非传染病并重；
  - 重视**人群健康**和生活质量研究；
  - 发展**应急**和**现场**流行病学；
  - 重视流行病学研究中的**伦理**学问题；
  - 强化流行病学在**循证实践**中的作用。
